# Dostaffkin

Небольшой SPA на Angular для расчёта стоимости доставки и отслеживания заявок.

## О проекте
Приложение позволяет пользователю указать адреса «откуда» и «куда», выбрать параметры (размер посылки, скорость) и получить стоимость и ориентировочное время доставки. После расчёта можно оформить заявку. Также доступна страница отслеживания по номеру заказа.

Интерактивная карта построена на API Яндекс.Карт (ymaps), адреса вводятся с автодополнением.
Данные заявки отправляются на тестовый сервер `https://testologia.ru/delivery`.

## Ключевой функционал
- Главная страница с переходом к оформлению заказа и трекингу.
- Форма расчёта маршрута с выбором размера (`xs`, `s`, `m`, `l`, `xl`) и скорости (`regular`, `fast`).
- Автоподстановка адреса по геолокации и подсказки от Яндекса.
- Отображение маршрута и расчёт стоимости/дистанции/времени.
- Отправка заявки через API и показ идентификатора заказа.
- Страница отслеживания статуса по номеру отправления.

## Технологический стек
- Язык: TypeScript
- Фреймворк: Angular 21
- UI: стандартные шаблоны Angular, CSS
- HTTP: встроенный `HttpClient`
- Карты: Яндекс.Карты (ymaps)
- Уведомления: `ngx-toastr`
- Реактивные формы: `@angular/forms`
- Управление состоянием: Angular signals
- Инструменты: `@angular/cli`, `vitest` (для тестов), `prettier`

## Установка и запуск
```bash
npm install       # установить зависимости
npm run start     # запустить dev-сервер (http://localhost:4200)
npm run build     # собрать приложение для продакшна
npm run test      # запустить тесты
```

> Требуется Node.js с npm (пакетный менеджер версия 10+ указана в package.json).
